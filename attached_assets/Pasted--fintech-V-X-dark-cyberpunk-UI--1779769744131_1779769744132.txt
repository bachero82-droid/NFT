Создай футуристическое fintech-приложение под названием “VΞX”.

Стиль:
- dark cyberpunk UI
- чёрный фон
- неоновые фиолетовые/синие градиенты
- glassmorphism
- плавные анимации
- премиальный футуристичный дизайн
- адаптивность для ПК и телефона
- смесь банка + NFT + игровой платформы

Технологии:
- Frontend: Next.js + TailwindCSS + Framer Motion
- Backend: Node.js + Express
- База данных: PostgreSQL
- ORM: Prisma
- Авторизация: JWT + bcrypt
- Real-time: Socket.io
- Хранение файлов: Supabase Storage или Cloudinary

Главная идея:
VΞX — цифровой банк нового поколения с NFT-картами, маркетплейсом, переводами между пользователями и встроенными играми.
